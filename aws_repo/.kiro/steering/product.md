# Product Overview

This is a hackathon project: a web application for managing an **autonomous greenhouse on Mars**.

Teams design and prototype an AI agent system capable of managing Martian agriculture — choosing suitable crops, monitoring environmental conditions, and optimizing resources.

A Mars Crop Knowledge Base is available via MCP server providing data on Mars environmental constraints, crop profiles (lettuce, potatoes, radishes, beans/peas, herbs), nutrient/water management, greenhouse operational scenarios, and more. Use it to inform features and UI.

The project is built during a time-boxed hackathon — favor working software over perfection. Start simple (basic CRUD), then layer on features.
